# PDF Metadata Writer

A simple Python script to add metadata to PDF files using the PyPDF2 library.

## Requirements
1. Python 3.x
2. PyPDF2 
 ```bash
 pip install PyPDF2
 ```

## Usage

1. Update the `input_pdf` and `output_pdf` paths in `pdf_metadata_writer.py`.
2. Run the script:

   ```bash
   python pdf_metadata_writer.py
   ```
